Beauty Products Sales Analysis — Excel Dashboard
This project analyzes sales data for 9 beauty products (Lipstick, Lip Balm, Lip Liner, Sunscreen, Cetaphil Moisturizer, Cetaphil Lotion, Sebamed Body Wash, Sebamed Lotion, and Shampoo) across two years — 2025 and 2026.
Tools Used: Microsoft Excel, SQL, Power BI
Key Findings:
Total Sales grew from $10,840 (2025) → $17,320 (2026) — a 59.8% increase
Net Profit jumped from $350 → $2,800 — a 700% growth
Total Loss reduced from $1,800 → $1,040 — a 42.2% improvement
Net Revenue increased by 78.6%
What this project covers:
Data cleaning and transformation
KPI tracking (Sales, Revenue, Profit, Returns, Loss)
Year-over-year comparison analysis
Product-level performance insights
Dashboard creation with interactive filters
